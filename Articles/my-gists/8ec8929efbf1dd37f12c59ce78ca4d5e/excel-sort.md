# SORT関数: Excelでデータをソートする新しい方法

Excelの最新バージョンでは、`SORT`関数が導入され、データをより柔軟にソートできるようになりました。この新しい関数は、従来の`SORT`関数よりも進化しており、様々なソート条件に対応しています。

## SORTの基本的な構文

`SORT`関数の基本的な構文は以下の通りです。

```excel
=SORT(配列, [ソートの列], [ソートの順序], [主キー1], [順序1], ...)
```
配列: ソート対象のデータが入っている範囲。
ソートの列: ソートを行う列の番号または範囲。
ソートの順序: 昇順（1）または降順（-1）を指定。
主キー1, 順序1, ...: ソートの優先順位が複数ある場合に指定。
SORTの利点
柔軟なソート条件: 複数のソート条件を指定できるため、より柔軟なソートが可能。
配列全体のソート: 特定の列だけでなく、行全体をソートできる。

```
=SORT(A2:C10, 2, -1, 1, 1)
```

この例では、A2からC10の範囲を、2列目（B列）で降順にソートし、その後1列目（A列）で昇順にサブソートしています。

## 注意点
動作環境: SORT関数はExcel 365およびExcel 2019以降でのみ利用可能。
ソートの列番号: ソートの列番号は範囲内である必要があります。
SORT関数は、より柔軟で強力なソート機能を提供し、大量のデータを効果的に整理できます。Excelの最新バージョンを利用している場合は、ぜひ活用してみてください。
