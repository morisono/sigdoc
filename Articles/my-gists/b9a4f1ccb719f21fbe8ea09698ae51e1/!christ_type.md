# キリスト教の主要な種類:

1. **カトリック教会**
   - ローマ教皇を教会の指導者とする。
   - 聖ペテロの後継者と信じられている。
   - 世界中に教会が広がっており、典礼や信仰体験に独自の要素がある。

2. **プロテスタント**
   - マルティン・ルター、ジョン・カルヴァン、ジョン・ウェスレーなどの宗教改革者によって始まった。
   - 聖書の中心的地位と信仰の個人的な解釈を重視する。
   - さまざまな教派や宗教共同体が存在し、信仰や儀式に多様性がある。

3. **オーソドックス教会**
   - 東方正教会とギリシャ正教会などが含まれる。
   - 早期教会の伝統を尊重し、宗教芸術や典礼に特徴がある。
   - 一神教信仰と三位一体を強調する。

4. **聖書信仰派（エヴァンジェリカル）**
   - 聖書を信仰の最高権威とする。
   - キリストの信仰告白と個人的な信仰の強調がある。
   - 福音主義教会やバプテスト教会などがこれに該当する。

5. **プロテスタントの主要な教派**
   - ルーテル教会、長老派教会、改革派教会、メソジスト教会など、さまざまな宗派が存在する。
   - 教義や礼拝形式、教会組織に差異がある。

6. **非トリニタリアン教会**
   - 三位一体論を否定する教派。
   - ユニタリアン教会などが含まれる。

7. **新宗教**
   - キリスト教の影響を受けつつも、独自の教義や実践を持つ宗教団体。
   - 代表的なものにモルモン教やジェホバの証人がある。

これは一般的なキリスト教の分類ですが、さらに多くの教派や宗教的なバリエーションが存在します。
