### PDFパスワード解析

   ```bash
   sudo apt-get install hashcat
   pdf2john yourfile.pdf > hash.txt
   hashcat -m 10500 -a 0 hash.txt wordlist.txt　
   # or 
   john --format=pdf <hashfile> --wordlist=<wordlist>
   ```
ハッシュを使用することで、平文のパスワードが漏洩するリスクが低減され、セキュリティが向上します。
