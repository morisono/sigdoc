# JTC DOC

[JTC DOC](https://theme.typora.io/doc/ja/Write-Custom-Theme/)（Japanese Traditional Company Documents style）テーマが新たにリリースされました。

この新しいテーマは、日本の伝統的な企業文書スタイルにインスパイアを受けてデザインされました。JTC DOCテーマは、伝統的な日本のビジネス文書を作成する際に、シンプルで洗練された書式設定を提供します。

新しいテーマの特徴には以下が含まれます：

- **美しい日本語フォント**: JTC DOCテーマでは 標準的な日本語フォントが採用されており、日本語の文章をより魅力的に表示します。

- **伝統的な書式**: テーマは、伝統的なビジネス文書のスタイルに合わせて段落、見出し、箇条書きなどの書式設定が調整されています。

- **使いやすさ**: Typoraの使いやすさと組み合わせて、日本語の文章を簡単に編集し、美しくフォーマットできます。

JTC DOCテーマは、日本の企業文書やレポートを作成するユーザー向けに設計されており、Typoraコミュニティのメンバーによるコラボレーションによって開発されました。

このテーマについての詳細やインストール方法については、[JTC DOCテーマのドキュメンテーション](https://theme.typora.io/doc/ja/Write-Custom-Theme/)をご覧ください。

新しいテーマをお楽しみいただければ幸いです！
